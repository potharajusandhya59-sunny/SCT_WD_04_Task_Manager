# 📝 Task Manager Web Application

## 📌 Project Overview

The Task Manager Web Application is a responsive and user-friendly web application developed as part of the **SkillCraft Technology Web Development Internship (Task-04)**.

This application enables users to efficiently organize their daily activities by creating, updating, completing, and deleting tasks through an intuitive and modern interface.

---

## 🚀 Features

- ➕ Add New Tasks
- ✏️ Edit Existing Tasks
- 🗑️ Delete Tasks
- ✅ Mark Tasks as Completed
- 📅 Add Due Date
- ⏰ Add Task Time
- 📱 Responsive Design
- 🎨 Modern Neon User Interface
- ⚡ Smooth Animations and Hover Effects

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 💡 How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open **index.html** in any modern web browser.
4. Start managing your tasks.
